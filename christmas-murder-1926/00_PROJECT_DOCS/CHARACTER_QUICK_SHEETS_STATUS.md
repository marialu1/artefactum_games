## üéØ Quick Sheet Format Standards

All quick sheets follow this structure:

### Page Structure (10 pages total):
- **PAGE 1**: Front Cover (PNG image)
- **PAGE 2**: Identity
- **PAGE 3**: Personality & Performance
- **PAGE 4**: Public Secret
- **PAGE 5**: Private Secrets (up to 2)
- **PAGE 6**: Private Secrets (continued, 2 more)
- **PAGE 7**: Goals + Did You Kill Him?
- **PAGE 8**: Relationships
- **PAGE 9**: How to Play
- **PAGE 10**: Back Cover (PNG image)

### Layout Principles:
- Keep related content together (no awkward pagebreaks mid-section)
- "How to Play" must fit on a single page
- ACT IV suicide/death instructions should be continuous
- Use `#pagebreak()` judiciously

### Design Standards:
- **Colors**: vintage-brown (#704214), vintage-black (#1a1a1a), gold (#D4AF37), cream (#FFFDF5)
- **Fonts**: "Great Vibes" (headings), "DejaVu Serif" (body), "Lato" (labels)
- **Required Assets**: Front cover PNG, Back cover PNG, Page number ornament

### Build Command:
```bash
typst compile <character>_quick.typ <CHARACTER>_QUICK.pdf
```
