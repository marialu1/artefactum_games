## üé≠ Game Master Instructions

### When to Distribute
- **Timing:** Hand out WITH character booklets at game start
- **Method:** Include in player pack envelopes OR attach to booklet
- **Visibility:** Each player only sees their OWN tier1 clue

### How Players Use These Clues
1. **Private Knowledge:** Players read these BEFORE game begins
2. **Establishes Motivation:** Gives character a reason to investigate
3. **Information Asymmetry:** Each player has unique starting information
4. **Trading Potential:** Players may choose to share/trade this info during game

### Printing Guidelines
- **Format:** Print at actual size (no scaling)
- **Quality:** High resolution for readability
- **Paper:** Standard printer paper acceptable
- **Handling:** Place in sealed envelope marked "CONFIDENTIAL - DO NOT OPEN UNTIL INSTRUCTED"
