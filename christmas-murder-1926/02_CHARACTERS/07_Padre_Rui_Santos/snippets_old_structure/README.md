# Father Juan Carlos - Snippet Files

This directory contains modular content snippets for **Father Juan Carlos** character materials.

## Purpose
These snippets enable:
- **Flexible content assembly** for different formats (booklets, quick sheets, Canva layouts)
- **Version control** for character development
- **Easy updates** - change once, propagate everywhere
- **Mix-and-match** content for various player materials

## File Structure

### Metadata
- `00_metadata.yaml` - Character name, player, age, role, status, archetype

### Core Identity
- `02_identity.md` - Basic facts, connection to host, core conflict
- `03_relationships.md` - Connections with other guests (color-coded)
- `04_personality.md` - Traits, appearance, mannerisms, speech patterns
- `05_goals.md` - Objectives for the night
- `06_did_you_kill.md` - Innocence/guilt status (INNOCENT - but knows who confessed)

### Secrets System
- `07_public_secret.md` - Info player can share freely (here to bless dinner, Reggie donated $500)
- `08_private_secret_1.md` - Sofia Mendes (victim of Reggie's predation, confessed to him)
- `09_private_secret_2.md` - Emilia Harrods is ex-nun Sister Cristina (personal rivalry)
- `10_private_secret_3.md` - Confession of murder (Dec 20, recognizes James's voice)

### Gameplay Guide
- `11_how_to_play.md` - Performance tips, key scenes, act-by-act strategy

## Usage

These snippets are designed to be:
1. **Source material** for full character dossiers
2. **Modular components** for Canva design layouts
3. **Reference documents** during character development
4. **Building blocks** for automated document generation
