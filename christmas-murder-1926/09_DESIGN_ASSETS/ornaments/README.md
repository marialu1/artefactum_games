## Required Ornaments (User to Export from Canva)

### divider_small.png
- **Size:** 20-30mm wide
- **Format:** PNG transparent background
- **Resolution:** 300 DPI minimum
- **Use:** Small decorative line between sections

### divider_medium.png
- **Size:** 40-50mm wide
- **Format:** PNG transparent background
- **Resolution:** 300 DPI minimum
- **Use:** Medium decorative element

### divider_tiny.png
- **Size:** 15-20mm wide
- **Format:** PNG transparent background
- **Resolution:** 300 DPI minimum
- **Use:** Tiny separator

### page_number_frame.png
- **Size:** 15mm × 8mm
- **Format:** PNG transparent background
- **Resolution:** 300 DPI minimum
- **Use:** Small frame for page numbers

## How to Export from Canva

1. Select ornamental element in Canva
2. Download → PNG transparent
3. Save to this directory
4. Rebuild Quick Sheet PDF
