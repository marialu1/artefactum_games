## Projects

### Orfanato Nossa Senhora das Dores
**Status:** In Development
**Type:** Hybrid physical + digital investigation game
**Players:** 4-10 (cooperative)
**Duration:** 2-3 hours

A supernatural murder mystery set in a Portuguese orphanage. Three nuns are found dead simultaneously - each killed in a different way. Players investigate as detectives, uncovering dark secrets about child abuse and spectral vengeance.
