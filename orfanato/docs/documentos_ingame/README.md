## Writing Guidelines

1. **Authenticity** - Documents should look and feel real
2. **Portuguese** - All content in European Portuguese
3. **Clues** - Each document serves the investigation
4. **Emotion** - Content should evoke player response
5. **No spoilers** - Each act only reveals what it should
