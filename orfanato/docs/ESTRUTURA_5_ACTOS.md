# ESTRUTURA DOS 5 ACTOS
## O Blueprint do Jogo

**Versao:** 1.0
**Data:** 8 Dezembro 2024
**Principio:** Menos documentos, mais impactantes. Cada peca SERVE para resolver algo.

---

## FILOSOFIA DE DESIGN

```
ANTES: 11 testemunhos + 16 personagens = jogador perde-se
AGORA: 6 testemunhos essenciais = jogador investido

O jogador e DETECTIVE, nao espectador.
A emocao vem de DESCOBRIR, nao de ser CONTADO.
```

**Duracao total:** 2-3 horas
**Promessa ao jogador:** "Descobre o que aconteceu."
**A verdade:** Nao foi assassinato humano. Foi justica sobrenatural. E as freiras mereciam.

## EXPERIENCIA MINUTO A MINUTO

### Minuto 0-5: Abertura
```
Jogador abre a caixa.
Ve: Pasta com "PROCESSO POLICIAL 2847/2024"
Sente: Curiosidade, expectativa
Accao: Abre envelope, le carta introdutoria
```

### Minuto 5-15: Contexto
```
Jogador le relatorio policial.
Descobre: 3 freiras mortas, mesma noite, orfanato em Sintra
Pensa: "Ok, assassino em serie?"
Accao: Procura mais detalhes
```

### Minuto 15-35: As Mortes
```
Jogador le autopsias.
Autopsia 1 - Conceicao: "Multiples impactos... como se atirada..."
Autopsia 2 - Beatriz: "Queimaduras sem fonte de calor..."
Autopsia 3 - Carmo: "Afogamento num balde de 30cm..."
Pensa: "Isto e IMPOSSIVEL"
Nota: Todas morreram as 03:33
Accao: Tenta codigo 0333
SUCESSO: Compartimento A abre
```

### Minuto 35-45: O Medico
```
Jogador le testemunho Dr. Antonio.
Descobre: "Nunca vi nada assim... as lesoes nao fazem sentido"
Confirma: Mortes sao sobrenaturais (mesmo que ele nao diga)
Pensa: "O que esta a acontecer neste orfanato?"
```

### Minuto 45-60: Os Abusos
```
Jogador le testemunhos Rosa, Ana.
Rosa: "As criancas tinham medo... castigos... a cela escura"
Ana: "Vejo uma menina... diz que se chama Sofia"
Pensa: "As freiras eram mas pessoas?"
Sente: Desconforto moral, a historia muda
```

### Minuto 60-75: A Testemunha
```
Jogador le testemunho Matilde.
Matilde: "Vi a Irma Leonor... foi puxada para CIMA"
Pensa: "Espera - uma CRIANCA viu o assassinato?"
Pensa: "E nao havia ninguem? Ela foi puxada por algo invisivel?"
Sente: Arrepio, confirmacao do sobrenatural
```

### Minuto 75-90: A Pista
```
Jogador le testemunho Helena.
Helena: "Seis letras... o rapaz que gostava de ler..."
Pensa: "Isto e uma charada"
Relembra: Autopsias mencionavam "Miguel" como ultimo orfao a morrer
Conta: M-I-G-U-E-L = 6 letras
Accao: Tenta codigo MIGUEL
SUCESSO: Compartimento B abre (arquivos)
```

### Minuto 90-110: As Criancas
```
Jogador le carta Sofia, diario Miguel.
Sofia: "Querida mae... tenho medo da agua... a Irma Carmo..."
Miguel: "Encontrei os registos... nao sao transferencias... SAO MORTES"
Joao: Registo de obito - "queda acidental" em 1973
Pensa: "Tres criancas mortas. Tres freiras mortas. Vinganca."
Sente: Horror, empatia, raiva
```

### Minuto 110-120: A Sobrevivente
```
Jogador le testemunho Carla.
Carla: "A Sofia salvou-me... empurrou-me para longe do poco..."
Carla: "Depois a Sofia caiu. Disseram que foi acidente."
Pensa: "Sofia MORREU a salvar outra crianca. E agora voltou."
Nota: Data no diario - 2008
Accao: Tenta codigo 2008
SUCESSO: Compartimento C abre
```

### Minuto 120-140: A Verdade
```
Jogador le registos de transferencias.
Ve: Nomes riscados - "transferido", "transferido", "transferido"
Percebe: Nao foram transferidos. Morreram.
Le carta de fundacao: "Inaugurado em 1960..."
Pensa: "64 anos. Quantas criancas?"
Accao: Tenta codigo 1960
SUCESSO: Compartimento Final abre
```

### Minuto 140-160: O Fecho
```
Jogador abre compartimento final.
Encontra: Boneca (Sofia), desenho queimado (Joao), livro gasto (Miguel)
Le carta final: A verdade completa.
App: Video de conclusao - as criancas finalmente descansam
Sente: Catarse. Paz. Justica foi feita.
Pensa: "E eu que descobri."
```

## EMOCOES POR ACTO

```
ACTO 1: Curiosidade → Confusao → Intrigo
        "O que aconteceu?" → "Isto e impossivel"

ACTO 2: Desconforto → Horror → Determinacao
        "As freiras eram mas" → "Preciso de saber mais"

ACTO 3: Empatia → Tristeza → Raiva
        "Pobres criancas" → "Isto e injusto"

ACTO 4: Compreensao → Ambivalencia → Aceitacao
        "Foi vinganca" → "Mas foi justica?"

ACTO 5: Reverencia → Paz → Satisfacao
        "Elas descansam" → "Eu resolvi isto"
```

## PROXIMOS PASSOS

Com esta estrutura definida:

1. **Escrever documentos por acto** (nao por tipo)
   - Comecar pelo Acto 1 completo
   - Testar em papel
   - Depois Acto 2, etc.

2. **Adaptar testemunhos existentes**
   - Ja temos 11 escritos
   - Seleccionar e editar os 6 essenciais
   - Cortar informacao redundante

3. **Criar documentos novos**
   - Autopsias (3)
   - Carta Sofia
   - Diario Miguel
   - Carta fundacao
   - Carta final
